# Sacred Space Ritual Calendar Layout Analysis

## Layout Structure:
- Header: "RITUALE" centered at top
- Search bar: "Rituale finden" with search icon
- Category dropdown: "KATEGORIE" with arrow
- Monthly sections with month name as separator
- Each ritual: circular photo thumbnail (left) + Title (bold) + Date (below title)
- Thin separator line between rituals
- Month headers in light gray text

## Key Design Elements:
- Clean, minimal list layout
- Circular thumbnails (photos, not emojis)
- Bold uppercase titles
- Date in format D.M.YYYY
- Monthly grouping with light separator
- Search + Category filter at top
