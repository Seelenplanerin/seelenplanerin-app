# Die Seelenplanerin – App Design

## Brand Identity

Die Seelenplanerin richtet sich an Frauen, die sich selbst wieder spüren wollen. Das Design spiegelt mystische Weiblichkeit, Mondenergie und spirituelle Tiefe wider – warm, einladend und transformativ.

## Screen List

1. **Home** – Tagesimpuls, Mondphase, Kategorien-Kacheln, Featured Content
2. **Entdecken** – Alle Inhalte, Filter nach Kategorie, Suche
3. **Rituale** – Rituale-Liste mit Kategorien (Mondrituale, Morgenrituale, Saisonrituale)
4. **Meditationen** – Geführte Meditationen mit Dauer-Anzeige
5. **Gedichte** – Poetische Texte, schön formatiert
6. **Impulse** – Tägliche Impulse und Affirmationen
7. **Favoriten** – Gespeicherte Inhalte
8. **Detail** – Universeller Detail-Screen für alle Content-Typen

## Navigation

Bottom Tab Bar mit 4 Tabs:
- **Seele** (Home) – house.fill
- **Entdecken** – sparkles (magnifyingglass)
- **Mond** – moon.fill
- **Favoriten** – heart.fill

## Key User Flows

1. Öffnen → Home → Tagesimpuls lesen → Kategorie wählen → Inhalt lesen/erleben
2. Entdecken → Filter → Detail-Screen → Favorit speichern
3. Mondphasen → Aktuelle Phase → Passende Rituale entdecken
4. Favoriten → Gespeicherte Inhalte → Erneut lesen

## Typography

- Headings: Serif-Stil (Georgia/System Serif) – mystisch, zeitlos
- Body: System Sans-Serif – lesbar, klar
- Quotes/Poems: Kursiv, zentriert – poetisch

## UI Components

- **Content Cards**: Abgerundete Karten mit Gradient-Overlay, Kategorie-Badge
- **Category Pills**: Horizontales Scroll-Menü für Filter
- **Moon Phase Widget**: Animierter Mond auf Home Screen
- **Tagesimpuls Banner**: Großes Hero-Element mit Zitat
- **Favoriten-Herz**: Animiertes Herz-Icon auf allen Cards
