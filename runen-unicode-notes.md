# Runen Unicode Verifizierung

Sowilo hat ZWEI Unicode-Varianten:
- ᛊ (U+16CA) = Sowilo S - ältere Form (wie griechisches Sigma)
- ᛋ (U+16CB) = Sowilo S - neuere Form (Blitz/Zickzack)

Im Code verwenden wir ᛊ (U+16CA) - das ist die ältere Elder Futhark Form.
Die bekanntere "Blitz"-Form wäre ᛋ (U+16CB).

Beide sind korrekt für Elder Futhark. Die Nutzerin meint vermutlich, dass ᛊ nicht wie die typische Sowilo-Rune aussieht, die man kennt (ᛋ).
